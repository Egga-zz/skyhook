skyhook
=======

Java Tools that elevate the starting point.


Purposes:
* Learning. Try to use some libraries / tools, e.g.
  * Gradle
  * hamsandwich
  * Gson
  * Travis CI
* Provide the simplest possible way to solve basic problems.
